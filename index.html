SELECTED PROJECTS

下面这些项目主要来自学习过程中的课程设计、实验和阶段性实践。

01

基于多模态数据的智能问答平台设计与实现

围绕复杂图文文档场景下的信息理解问题，完成了一个基于 VLM + RAG 的多模态问答平台设计与实现。

  • 完成文档解析、知识库构建、混合检索与回答生成等核心模块开发
  • 基于 Qwen2.5-VL 实现版面结构识别、Markdown 转换、图片裁剪与 Caption 生成
  • 构建图文联合索引与图片召回增强机制，支持文本检索、关键词检索、关联图片展示
  • 构建 groundtruth 数据集并完成回答质量评估与系统测试

02

基于 MoCo 的人脸验证模型

结合课程学习内容，尝试用 MoCo + Triplet Loss 的方法完成人脸验证模型训练与效果验证。

  • 基于 MTCNN 完成人脸预处理与数据流程搭建
  • 设计两阶段训练策略，并结合 semi-hard triplet 进行优化
  • ResNet-50 在 LFW 上取得 85.1% Accuracy、0.844 AUC、0.821 F1-score

03

基于 LangGraph 与 LLM 的链路智能规划系统

围绕链路参数优化这一问题，尝试搭建基于 LangGraph + LLM 的链路智能规划系统。

  • 设计需求解析、环境感知、规划、执行、观测、反思的 PAOR Agent 工作流
  • 支持带宽、天线数、载波频率、MCS 等参数的动态优化
  • 在 64 个测试用例上完成 benchmark，系统取得 87.50% 达标率与 79.55 平均效用
